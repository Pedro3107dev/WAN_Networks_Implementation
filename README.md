## Author: Pedro Mauri Martínez - A01029143

## Abstract

This repository is part of my **"Last Round"** collection, which stores all the notes and exercises I completed during my last semester of university.

In this case, it's from the course **"Implementation of Wide Area Networks and Distributed Services"**, which primarily covered network security, WAN protocols, datacenters, WLCs, VPNs, and cloud connectivity.
